# Documentação Específica — Vertical Contábil

Esta pasta armazena documentação **específica do domínio Contábil** (Contábil Cloud, Planejamento, Tesouraria, Prestação de Contas, Portal do Gestor/Transparência). Para documentação compartilhada com outras verticais (incidentes da plataforma, padrões gerais), veja `../../docs/`.

## Conteúdo sugerido

- **Glossário técnico** — vocabulário específico (PCASP, MSC, SICONFI, RREO, RGF, DCA, restos a pagar, fonte/destinação de recursos, superávit financeiro etc.).
- **Exemplos de chamados** — exemplos anonimizados de chamados-padrão e seus comentários internos esperados.
- **Regras de negócio** — anotações sobre Lei 4.320/64, LRF, MCASP/STN, limites constitucionais, encerramento/abertura de exercício.
- **Mapeamento de produtos** — relação entre os produtos (Contábil, Planejamento, Tesouraria, Prestação de Contas) e seus principais módulos/erros conhecidos.
- **e-Sfinge / TCE-SC** — validações do módulo contábil, CONs recorrentes, anexos e demonstrativos com divergências comuns.
- **Integrações** — Folha x Contabilidade (CON808 e correlatas), Arrecadação x Contabilidade, OBN/pagamentos eletrônicos.
- **Decisões de projeto** — registros (ADRs) sobre mudanças de comportamento da triagem ao longo do tempo, específicas desta vertical.

> Esta pasta é versionada no Git e seu conteúdo serve como fonte de consulta para a IA e para os analistas.
