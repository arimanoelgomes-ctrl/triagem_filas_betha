# Documentação Específica — Vertical Saúde

Esta pasta armazena documentação **específica do domínio Saúde Municipal/SUS**. Para documentação compartilhada com outras verticais (incidentes da plataforma, padrões gerais), veja `../../docs/`.

## Conteúdo sugerido

- **Glossário técnico** — vocabulário específico (PEC, e-SUS APS, BPA, RAAS, FAA, CADSUS, CNS, CNES, CBO, SIGTAP, CID etc.).
- **Exemplos de chamados** — exemplos anonimizados de chamados-padrão e seus comentários internos esperados.
- **Regras de negócio** — anotações sobre portarias do Ministério da Saúde, regras de faturamento SIA/SUS, validações Datasus, regras de imunização, regras de regulação etc.
- **Mapeamento de produtos** — relação entre os produtos da Betha na vertical Saúde e seus principais módulos/erros conhecidos.
- **Procedimentos SIGTAP** — referência das regras de cobertura/incompatibilidade mais recorrentes.
- **Validações do Datasus** — códigos de erro do BPA/RAAS/PEC e seus tratamentos validados.
- **Decisões de projeto** — registros (ADRs) sobre mudanças de comportamento da triagem ao longo do tempo, específicas desta vertical.

> Esta pasta é versionada no Git e seu conteúdo serve como fonte de consulta para a IA e para os analistas.
