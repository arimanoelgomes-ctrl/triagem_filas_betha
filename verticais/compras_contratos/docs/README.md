# Documentação Específica — Vertical Compras/Contratos

Esta pasta armazena documentação **específica do domínio Compras/Contratos** (Compras Cloud, Contratos Cloud, licitações, PNCP, envio de atos de contratação ao TCE-SC). Para documentação compartilhada com outras verticais (incidentes da plataforma, padrões gerais), veja `../../docs/`.

## Conteúdo sugerido

- **Glossário técnico** — vocabulário específico (NLLC, pregão, dispensa, inexigibilidade, ata de registro de preços, adesão/carona, aditivo, apostilamento, reequilíbrio etc.).
- **Exemplos de chamados** — exemplos anonimizados de chamados-padrão e seus comentários internos esperados.
- **Regras de negócio** — anotações sobre Lei 14.133/2021, Lei 8.666/93, decretos regulamentadores, prazos e modalidades.
- **Mapeamento de produtos** — relação entre os produtos (Compras, Contratos) e seus principais módulos/erros conhecidos.
- **e-Sfinge / TCE-SC e PNCP** — validações de atos de contratação, erros recorrentes de envio, comunicados do TCE-SC, regras de publicação no PNCP.
- **Integrações** — Compras/Contratos x Contabilidade (empenhos), Contratos x Betha Saúde (vínculo de contratos), demais consumidores.
- **Decisões de projeto** — registros (ADRs) sobre mudanças de comportamento da triagem ao longo do tempo, específicas desta vertical.

> Esta pasta é versionada no Git e seu conteúdo serve como fonte de consulta para a IA e para os analistas.
