# Documentação Específica — Vertical Pessoal

Esta pasta armazena documentação **específica do domínio Pessoal** (Folha, eSocial, Ponto, RH). Para documentação compartilhada com outras verticais (incidentes da plataforma, padrões gerais), veja `../../docs/`.

## Conteúdo sugerido

- **Glossário técnico** — vocabulário específico (rubricas, eventos eSocial, fechamento, jornada, banco de horas, vínculos, lotações etc.).
- **Exemplos de chamados** — exemplos anonimizados de chamados-padrão e seus comentários internos esperados.
- **Regras de negócio** — anotações sobre CLT, INSS, FGTS, IRRF, salário-família, salário-maternidade, FAP/RAT e regras municipais de estatutários recorrentes.
- **Mapeamento de produtos** — relação entre os produtos (Folha Cloud, eSocial, Minha Folha, Ponto Cloud, Pontual Cloud, Recursos Humanos Cloud) e seus principais módulos/erros conhecidos.
- **Eventos eSocial** — referência dos eventos S-1xxx, S-2xxx, S-3xxx, S-5xxx e seus retornos/inconsistências mais comuns.
- **Decisões de projeto** — registros (ADRs) sobre mudanças de comportamento da triagem ao longo do tempo, específicas desta vertical.

> Esta pasta é versionada no Git e seu conteúdo serve como fonte de consulta para a IA e para os analistas.
